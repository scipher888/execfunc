# Contributing to ExecFunc

Thanks for your interest! ExecFunc is in early development — we're building and testing with one family before opening contributions.

## Current Status

**Not yet accepting contributions.** The framework needs to stabilize first.

## When We're Ready

When contributions open, these areas would be most valuable:

- Additional OpenClaw skills for task management
- Integration patterns for different calendars, messaging apps, and tools
- Accessibility improvements
- Evidence and research pointers
- Translations

## How to Follow Along

- **Star this repo** to get notified of updates
- **Read the docs** in the `docs/` folder to understand the design philosophy
- **Open an issue** if you find a bug or have a suggestion — we're reading everything even if we can't act on it yet

## Security Issues

Please report security issues privately to [security@execfunc.ai](mailto:security@execfunc.ai) rather than opening a public issue.

## Code of Conduct

Be kind. This is a family project built for people who are struggling. We expect the same empathy in our community that we design into our product.
